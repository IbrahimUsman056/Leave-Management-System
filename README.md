Leave Management System

The Leave Management System is a full-featured web application for managing employee leave requests efficiently. It demonstrates secure authentication, role-based access control, and CRUD operations for both employees and leave requests.

Features

Authentication & Authorization: Uses ASP.NET Core Identity for secure login and registration.

Role-Based Access Control:

Admin: Manage employees, approve/reject leave requests.

Employee: Apply for leave, view status of own requests.

Leave Management: Employees can apply for casual, sick, annual, or emergency leaves.

Responsive UI: Bootstrap-based interface for easy navigation on desktop and mobile.

Notifications: Displays success/error messages for user actions.

Technologies Used

ASP.NET Core MVC

Entity Framework Core

ASP.NET Core Identity

C#

Bootstrap 5

SQL Server
